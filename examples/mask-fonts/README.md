# Mask fonts

Font data for [`examples/amazing.trx`](../amazing.trx)'s masking feature
(`--mask-text` / `--mask logo` / `--font`). Each `*.trx` file registers one
font into the `MASK-FONTS` registry and is loaded lazily on first use.

Trix has no FreeType, so fonts cannot be rasterised inside the VM. Instead a
host tool pre-renders each font into Trix data here — **only the derived glyph
data is committed, never a `.ttf`.**

| File | `--font` | Kind | Source / license |
| --- | --- | --- | --- |
| `roboto-bold.trx` | `roboto-bold` (default) | bitmap atlas | Roboto Bold — **Apache-2.0** |
| `roboto-mono-bold.trx` | `roboto-mono-bold` | bitmap atlas | Roboto Mono Bold — **Apache-2.0** |
| `hershey-sans.trx` | `hershey-sans` | stroke | Hershey Roman Simplex — **public domain** |
| `hershey-serif.trx` | `hershey-serif` | stroke | Hershey Roman Triplex — **public domain** |

(`--font 5x7` is a built-in block font inside `amazing.trx` and needs no file.)

- **Bitmap atlases** are per-glyph packed bitmaps on a common baseline; the maze
  fills the letter bodies. Generated by [`tools/gen_mask_font.py`](../../tools/gen_mask_font.py)
  (needs Pillow + NumPy; downloads Roboto).
- **Stroke fonts** are polyline centerlines rendered in pure Trix by stroking
  each segment with a pen width (vector → crisp at any size; engraved look).
  Re-encoded from the public-domain Hershey coordinate designs by
  [`tools/gen_hershey_font.py`](../../tools/gen_hershey_font.py) (standard library only).

## Regenerate / add a font

```bash
# the two bundled families
tools/gen_mask_font.py          # roboto-bold.trx + roboto-mono-bold.trx
tools/gen_hershey_font.py       # hershey-sans.trx + hershey-serif.trx

# any font you have, for your own local mazes (not committed)
tools/gen_mask_font.py --ttf /path/to/Some-Bold.ttf --name some-bold
./trix examples/amazing.trx --mask-text Hello --font some-bold --out hi.png
```

## Licensing

Informational, not legal advice — verify before relying. In the US a typeface
*design* (and the pixels it renders) is not copyrightable; only the font
*program* is, and the font *name* may be a trademark. So rendering glyphs for
**your own output** is unencumbered regardless of the source font's license —
you are producing pixels, not shipping the font. The license only matters if you
intend to **commit / redistribute** the generated atlas.

This project bundles only **Apache-2.0** (Roboto) and **public-domain** (Hershey)
glyph data. `tools/gen_mask_font.py` carries a license-by-font table; the root
[`NOTICE.md`](../../NOTICE.md) holds the attributions. OFL (Liberation/Noto, most
Google Fonts) and Bitstream Vera (DejaVu) add restrictions beyond Apache-2.0
(reserved names, no-sale-standalone), so their derived data is **not** committed
here — use `--ttf` to render them locally instead.
